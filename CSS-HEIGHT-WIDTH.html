CSS Height and Width Properties

CSS height and width properties are used to set the dimensions of elements. These properties determine how much space an element takes up on the page.

Property Description
height Sets the height of an element
width Sets the width of an element
max-height Sets the maximum height of an element
max-width Sets the maximum width of an element
min-height Sets the minimum height of an element
min-width Sets the minimum width of an element

Setting Height and Width

The height and width properties can be set using various units:

div {
    height: 200px;
    width: 50%;
}

.box {
    height: 10em;
    width: 20em;
}

Maximum Width

The max-width property sets the maximum width of an element. This is useful for creating responsive designs:

div {
    max-width: 500px; /* Element will never be wider than 500px */
    width: 100%; /* Element will take 100% of the available width, up to 500px */
}

Minimum Height

The min-height property ensures an element is at least a certain height:

div {
    min-height: 200px; /* Element will be at least 200px tall */
}

Height and Width Units

CSS dimensions can be specified using various units:

/* Pixels - fixed units */
div.fixed {
    height: 100px;
    width: 200px;
}

/* Percentage - relative to parent element */
div.percent {
    height: 50%;
    width: 75%;
}

/* em - relative to font-size */
div.em {
    height: 10em; /* 10 times the element's font size */
    width: 20em;
}

/* viewport units */
div.viewport {
    height: 50vh; /* 50% of the viewport height */
    width: 80vw; /* 80% of the viewport width */
}

Height and Width with Box-Sizing

By default, the width and height properties set the size of the content area. If you want these properties to include padding and border, use box-sizing: border-box:

div {
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    box-sizing: border-box; /* Total width will be 300px including padding and border */
}
