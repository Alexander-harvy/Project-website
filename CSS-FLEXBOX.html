CSS Flexbox Layout

Flexbox (Flexible Box Layout) is a one-dimensional layout method designed for laying out items in rows or columns. It provides a more efficient way to distribute space and align items within a container, even when their size is unknown or dynamic.

Creating a Flex Container

To use flexbox, you first need to define a flex container by setting the display property to flex or inline-flex:

.container {
    display: flex; /* or inline-flex */
}

Flex Container Properties

These properties are set on the parent element (the flex container):

Property Description
flex-direction Specifies the direction of the flexible items (row, row-reverse, column, column-reverse)
flex-wrap Specifies whether flex items should wrap or not (nowrap, wrap, wrap-reverse)
flex-flow Shorthand property for flex-direction and flex-wrap
justify-content Aligns flex items along the main axis (flex-start, flex-end, center, space-between, space-around, space-evenly)
align-items Aligns flex items along the cross axis (flex-start, flex-end, center, baseline, stretch)
align-content Aligns flex lines when there is extra space in the cross-axis (flex-start, flex-end, center, space-between, space-around, stretch)

Flex Item Properties

These properties are set on the children elements (the flex items):

Property Description
order Controls the order of flex items in the container
flex-grow Specifies how much a flex item will grow relative to other flex items
flex-shrink Specifies how much a flex item will shrink relative to other flex items
flex-basis Specifies the initial length of a flex item
flex Shorthand property for flex-grow, flex-shrink, and flex-basis
align-self Overrides the align-items value for specific flex items

Common Flexbox Layouts

Here are some examples of common layouts using flexbox:

Center Alignment (Horizontally and Vertically)

.container {
    display: flex;
    justify-content: center;    /* Horizontal centering */
    align-items: center;        /* Vertical centering */
    height: 300px;              /* Container height */
}

Navigation Bar

.navbar {
    display: flex;
    justify-content: space-between;  /* Items spaced evenly */
}

.nav-links {
    display: flex;
    gap: 20px;                      /* Space between items */
}

Card Layout

.card-container {
    display: flex;
    flex-wrap: wrap;                 /* Allow items to wrap */
    gap: 20px;                       /* Space between cards */
}

.card {
    flex: 1 0 300px;                 /* Grow, don't shrink, base width */
    max-width: 400px;                /* Maximum width */
}

Flexbox vs. Float

Flexbox offers several advantages over the traditional float property:

  • Direction-agnostic: Works in both horizontal and vertical directions
  • Alignment capabilities: Easy alignment of items (center, end, space-between, etc.)
  • Even distribution: Space can be distributed evenly among items
  • Reordering: Items can be visually reordered without changing HTML
  • No clearfix: No need for clearfix hacks as with floats

Browser Support

Flexbox is supported in all modern browsers, including:

  • Chrome 29+
  • Firefox 22+
  • Safari 6.1+
  • Edge (all versions)
  • Opera 12.1+
  • iOS Safari 7.1+
  • Android Browser 4.4+
