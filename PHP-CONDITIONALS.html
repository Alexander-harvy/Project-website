PHP Conditional Statements

Conditional statements are used to perform different actions based on different conditions.

In PHP, we have the following conditional statements:

  • if statement - executes code if one condition is true
  • if...else statement - executes one code if a condition is true and another code if that condition is false
  • if...elseif...else statement - executes different codes for more than two conditions
  • switch statement - selects one of many blocks of code to be executed

The if Statement

The if statement executes a block of code if a specified condition is true.


<?php
// Syntax
if (condition) {
    // code to be executed if condition is true
}

// Example
$t = 15;

if ($t < 20) {
    echo "Have a good day!";
}
// Outputs: "Have a good day!"
?>

The if...else Statement

The if...else statement executes one block of code if a condition is true and another block of code if the condition is false.


<?php
// Syntax
if (condition) {
    // code to be executed if condition is true
} else {
    // code to be executed if condition is false
}

// Example
$t = 25;

if ($t < 20) {
    echo "Have a good day!";
} else {
    echo "Have a good night!";
}
// Outputs: "Have a good night!"
?>

The if...elseif...else Statement

The if...elseif...else statement executes different blocks of code for more than two conditions.


<?php
// Syntax
if (condition1) {
    // code to be executed if condition1 is true
} elseif (condition2) {
    // code to be executed if condition1 is false and condition2 is true
} else {
    // code to be executed if both condition1 and condition2 are false
}

// Example
$t = 22;

if ($t < 10) {
    echo "Have a good morning!";
} elseif ($t < 20) {
    echo "Have a good day!";
} else {
    echo "Have a good night!";
}
// Outputs: "Have a good night!"
?>

You can have as many elseif statements as you need, for example:


<?php
$t = 13;

if ($t < 10) {
    echo "Have a good morning!";
} elseif ($t < 15) {
    echo "Have a good afternoon!";
} elseif ($t < 20) {
    echo "Have a good evening!";
} else {
    echo "Have a good night!";
}
// Outputs: "Have a good afternoon!"
?>

Nested if Statements

You can place if statements inside other if statements to create more complex conditions.


<?php
$t = 25;
$weather = "sunny";

if ($t > 20) {
    echo "It's a warm day. ";
    
    if ($weather == "sunny") {
        echo "And it's sunny outside!";
    } else {
        echo "But it's not sunny.";
    }
}
// Outputs: "It's a warm day. And it's sunny outside!"
?>

Short Hand If...Else (Ternary Operator)

The ternary operator is a shorthand for the if...else statement.


<?php
// Syntax
variable = (condition) ? expressionIfTrue : expressionIfFalse;

// Example
$t = 15;
$result = ($t < 20) ? "Good day" : "Good night";
echo $result; // Outputs: "Good day"

// Without a variable
echo ($t < 20) ? "Good day" : "Good night";
// Outputs: "Good day"
?>

Null Coalescing Operator

The null coalescing operator (??) was introduced in PHP 7 as a shorthand for a common use case of the ternary operator. It returns the first operand if it exists and is not NULL, otherwise it returns the second operand.


<?php
// Syntax
$variable = $firstExpression ?? $secondExpression;

// Example
// Old way (ternary)
$username = isset($_GET['user']) ? $_GET['user'] : 'Guest';

// New way (null coalescing)
$username = $_GET['user'] ?? 'Guest';

// Can be chained
$username = $_GET['user'] ?? $_POST['user'] ?? 'Guest';
?>

Real-World Examples

Example 1: User Authentication


<?php
$username = "admin";
$password = "password123";

// Check if credentials are correct
if ($username == "admin" && $password == "password123") {
    echo "Login successful. Welcome Admin!";
} else {
    echo "Invalid username or password. Please try again.";
}
// Outputs: "Login successful. Welcome Admin!"
?>

Example 2: Grading System


<?php
$score = 85;
$grade = "";

if ($score >= 90) {
    $grade = "A";
} elseif ($score >= 80) {
    $grade = "B";
} elseif ($score >= 70) {
    $grade = "C";
} elseif ($score >= 60) {
    $grade = "D";
} else {
    $grade = "F";
}

echo "Your score is $score, which is grade $grade.";
// Outputs: "Your score is 85, which is grade B."
?>

Example 3: Checking Age for Different Content


<?php
$age = 17;

if ($age < 13) {
    echo "Content for children.";
} elseif ($age >= 13 && $age < 18) {
    echo "Content for teenagers.";
} else {
    echo "Content for adults.";
}
// Outputs: "Content for teenagers."
?>

Best Practices for Conditional Statements

  • Always use curly braces {} for if statements, even if they contain only one line of code. This improves readability and prevents errors.
  • When comparing variables against values, it's recommended to use the identical operator (===) instead of the equality operator (==) when possible to avoid type conversion issues.
  • Avoid deeply nested if statements as they make code hard to read and maintain. Consider refactoring with early returns or separate functions.
  • Use meaningful variable names that represent what you're checking for better code self-documentation.
  • For multiple complex conditions, consider storing them in separate variables with descriptive names.
