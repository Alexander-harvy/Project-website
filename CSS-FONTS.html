CSS Font Properties

CSS font properties define the font family, size, weight, style, and other aspects of text on a webpage. Fonts play a crucial role in the readability and overall aesthetic of your website.

Property Description
font-family Specifies the font for text
font-size Specifies the size of the font
font-weight Specifies the weight or thickness of a font
font-style Specifies the style of the font (normal, italic, oblique)
font-variant Specifies whether text should be displayed in small-caps
font Shorthand property for setting all font properties
@font-face Rule to define and use custom fonts

Font Family

The font-family property specifies the font for text. It can list multiple font names as a "fallback" system:

h1 {
    font-family: "Times New Roman", Times, serif;
}

p {
    font-family: Arial, Helvetica, sans-serif;
}

div {
    font-family: "Courier New", Courier, monospace;
}

Font Size

The font-size property sets the size of the text:

h1 {
    font-size: 40px;
}

h2 {
    font-size: 2em; /* 2 times the size of the current font */
}

p {
    font-size: 16px;
}

span {
    font-size: 100%; /* 100% of the parent element's font size */
}

div {
    font-size: 1.2rem; /* Relative to the root element's font size */
}

Font Weight

The font-weight property specifies the weight or thickness of a font:

h1 {
    font-weight: bold; /* or 700 */
}

h2 {
    font-weight: normal; /* or 400 */
}

p {
    font-weight: 300; /* Light */
}

span {
    font-weight: 900; /* Extra Bold */
}

Font Style

The font-style property specifies the style of the font:

h1 {
    font-style: normal; /* Default */
}

h2 {
    font-style: italic;
}

p {
    font-style: oblique;
}

Font Shorthand

The font property is a shorthand property for setting all font properties in one declaration:

p {
    /* Order: font-style font-variant font-weight font-size/line-height font-family */
    font: italic small-caps bold 16px/1.5 Arial, sans-serif;
}

Web Safe Fonts

Web safe fonts are fonts that are commonly available across different operating systems:

  • Arial, Helvetica, sans-serif
  • "Times New Roman", Times, serif
  • "Courier New", Courier, monospace
  • Georgia, serif
  • Verdana, Geneva, sans-serif

Google Fonts

Google Fonts provides a free library of web fonts that can be easily integrated into your website:

/* In the head section of your HTML */
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">

/* In your CSS */
body {
    font-family: 'Roboto', sans-serif;
}
