PHP Numbers

In PHP, numbers can be represented as integers or floating-point numbers (floats). PHP has several functions and features for working with numbers.

PHP Integers

An integer is a whole number (without decimals) that can be either positive or negative.


<?php
$x = 5985;      // positive integer
$y = -345;      // negative integer
$z = 0x8C;      // hexadecimal number
$a = 047;       // octal number
$b = 0b1010;    // binary number

var_dump($x);
var_dump($y);
var_dump($z);
var_dump($a);
var_dump($b);
?>

PHP Floats

A float (floating point number) is a number with a decimal point or a number in exponential form.


<?php
$x = 10.365;
$y = 2.4e3;
$z = 8E-5;

var_dump($x);
var_dump($y);
var_dump($z);
?>

Number Functions

PHP has several useful functions for working with numbers:

is_int() - Check if variable is integer


<?php
$x = 5985;
var_dump(is_int($x)); // bool(true)

$y = 59.85;
var_dump(is_int($y)); // bool(false)
?>

is_float() - Check if variable is float


<?php
$x = 10.365;
var_dump(is_float($x)); // bool(true)
?>

is_numeric() - Check if variable is numeric


<?php
$x = 5985;
var_dump(is_numeric($x)); // bool(true)

$y = "5985";
var_dump(is_numeric($y)); // bool(true)

$z = "59.85" + 100;
var_dump(is_numeric($z)); // bool(true)

$a = "Hello";
var_dump(is_numeric($a)); // bool(false)
?>

Type Casting

You can cast strings and floats to integers:


<?php
// Cast float to int
$x = 23465.768;
$int_cast = (int)$x;
echo $int_cast; // 23465

// Cast string to int
$y = "23465.768";
$int_cast = (int)$y;
echo $int_cast; // 23465
?>

Number Formatting

The number_format() function formats a number with grouped thousands:


<?php
echo number_format("1000000")."<br>";          // 1,000,000
echo number_format("1000000",2)."<br>";       // 1,000,000.00
echo number_format("1000000",2,",",".");      // 1.000.000,00
?>

Random Numbers

The rand() function generates a random number:


<?php
echo rand();           // random number between 0 and getrandmax()
echo rand(10, 100);    // random number between 10 and 100
?>

Practical Example: Shopping Cart

Here's an example of how numbers are used in a shopping cart calculation:


<!DOCTYPE html>
<html>
<head>
    <title>Shopping Cart</title>
    <style>
        table { border-collapse: collapse; width: 50%; }
        th, td { border: 1px solid #ddd; padding: 8px; text-align: left; }
        th { background-color: #f2f2f2; }
    </style>
</head>
<body>
    <?php
    // Define products with prices
    $products = array(
        array("name" => "Laptop", "price" => 999.99, "quantity" => 2),
        array("name" => "Mouse", "price" => 19.99, "quantity" => 3),
        array("name" => "Keyboard", "price" => 49.99, "quantity" => 1)
    );
    
    // Calculate subtotal, tax, and total
    $subtotal = 0;
    foreach ($products as $product) {
        $subtotal += $product["price"] * $product["quantity"];
    }
    $tax = $subtotal * 0.08; // 8% tax
    $total = $subtotal + $tax;
    ?>
    
    <h2>Shopping Cart</h2>
    <table>
        <tr>
            <th>Product</th>
            <th>Price</th>
            <th>Quantity</th>
            <th>Total</th>
        </tr>
        <?php foreach ($products as $product): ?>
        <tr>
            <td><?php echo $product["name"]; ?></td>
            <td>$<?php echo number_format($product["price"], 2); ?></td>
            <td><?php echo $product["quantity"]; ?></td>
            <td>$<?php echo number_format($product["price"] * $product["quantity"], 2); ?></td>
        </tr>
        <?php endforeach; ?>
        <tr>
            <td colspan="3">Subtotal</td>
            <td>$<?php echo number_format($subtotal, 2); ?></td>
        </tr>
        <tr>
            <td colspan="3">Tax (8%)</td>
            <td>$<?php echo number_format($tax, 2); ?></td>
        </tr>
        <tr>
            <td colspan="3"><strong>Total</strong></td>
            <td><strong>$<?php echo number_format($total, 2); ?></strong></td>
        </tr>
    </table>
</body>
</html>
