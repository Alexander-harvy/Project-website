HTML Images - Adding Visual Content to Your Web Pages

Images are essential elements in web design that enhance the visual appeal and user experience of your website. HTML provides a simple way to embed images through the <img> tag. This tutorial will guide you through the process of adding and manipulating images in your web pages.

Basic Image Syntax

The HTML image tag is a self-closing tag used to embed images:

<img src="image.jpg" alt="Description of image">

The most important attributes of the <img> tag are:

  • src - specifies the path to the image file
  • alt - provides alternative text for screen readers
  • width/height - specifies dimensions
  • title - shows a tooltip when hovering

Image File Types

Common image formats used on the web:

  • JPEG (.jpg, .jpeg) - Best for photographs
  • PNG (.png) - Best for graphics with transparency
  • GIF (.gif) - Supports animation, limited colors
  • SVG (.svg) - Vector graphics that scale without quality loss
  • WebP (.webp) - Modern format with better compression

Image Size & Positioning

Control image dimensions and alignment:

<!-- Setting width and height -->
<img src="logo.png" alt="Logo" width="300" height="200">

<!-- CSS styling for images -->
<img src="photo.jpg" alt="Photo" style="width:100%; max-width:500px;">

<!-- Centered image using CSS -->
<img src="banner.jpg" alt="Banner" style="display:block; margin:0 auto;">

Responsive Images

Making images adapt to different screen sizes is crucial for modern web design:

<!-- Basic responsive image -->
<img src="responsive.jpg" alt="Responsive Image" style="width:100%; height:auto;">

<!-- Using srcset for different resolutions -->
<img src="image-small.jpg" 
     srcset="image-small.jpg 500w,
             image-medium.jpg 1000w,
             image-large.jpg 1500w"
     sizes="(max-width: 600px) 500px,
            (max-width: 1200px) 1000px,
            1500px"
     alt="Responsive image with multiple sources">

Image Maps

Image maps allow you to define clickable areas on a single image:

<img src="workplace.jpg" alt="Workplace" usemap="#workmap">

<map name="workmap">
  <area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.html">
  <area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.html">
  <area shape="circle" coords="337,300,44" alt="Coffee cup" href="coffee.html">
</map>

Image as a Link

You can use images as clickable links by placing them inside an anchor tag:

<a href="https://www.example.com">
  <img src="button.jpg" alt="Visit Example.com">
</a>

HTML Image Attributes

Attribute Description Example
src Specifies the path to the image file src="images/photo.jpg"
alt Provides alternative text for screen readers alt="Mountain landscape"
width Specifies the width of the image (in pixels or %) width="500" or width="100%"
height Specifies the height of the image (in pixels or %) height="300" or height="50%"
loading Instructs the browser on how to load the image loading="lazy"
title Shows additional information as tooltip on hover title="Beautiful mountain view"
usemap References a map element to create an image map usemap="#mapname"

Best Practices for Web Images

  • Always include descriptive alt text for accessibility
  • Optimize images for web to improve page load speed
  • Specify width and height attributes to prevent layout shifts
  • Use responsive images to support various screen sizes
  • Choose the appropriate file format for your image type
  • Consider lazy loading for images below the fold
  • Maintain aspect ratio when resizing images
  • Use appropriate compression to balance quality and file size
