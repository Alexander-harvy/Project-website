HTML Comments - Hidden Notes in Your Code

HTML comments are special text blocks that are not displayed in the browser but can help document your HTML code. They are useful for explaining complex code sections, leaving notes for other developers, or temporarily disabling parts of your code without deleting it.

Comment Syntax

HTML comments start with <!-- and end with -->.

Example:

<!-- This is a comment -->

The comment above will not be visible when the page is displayed in a browser.

Uses of Comments

  • Document your code for future reference
  • Leave notes for other developers
  • Temporarily disable code without deleting it
  • Add reminders for future updates
  • Organize large HTML documents with section markers

Best Practices

When using HTML comments, follow these guidelines:

  • Keep comments concise and relevant
  • Don't include sensitive information in comments
  • Use consistent comment styles across your project
  • Consider removing unnecessary comments before production

Comment Examples

<!-- Basic comment -->
<p>This paragraph is visible
 to users.</p>

<!-- 
    Multi-line comment
    for longer explanations
    or temporary code removal
-->

<!-- <p>This paragraph is
 commented out and won't display.
 </p> -->
