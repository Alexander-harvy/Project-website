HTML Links - Connecting Your Web Pages

Links are an essential part of the web, allowing users to navigate between web pages and resources. They are the foundation of what makes the web a "web" - a network of interconnected documents and media. In HTML, links are created using the <a> (anchor) tag.

Basic Link Syntax

The HTML anchor tag is used to create hyperlinks:

<a href="url">Link Text</a>

The most important attributes of the <a> tag are:

  • href - specifies the destination URL
  • target - specifies where to open the link
  • title - provides additional information about the link
  • rel - specifies the relationship to the target

Link Targets

The target attribute specifies where to open the linked document:


<a href="https://www.example.com" target="_blank">
Opens in new tab</a>

<a href="about.html" target="_self">
Opens in same frame (default)</a>

<a href="services.html" target="_parent">
Opens in parent frame</a>

<a href="contact.html" target="_top">
Opens in full body of window</a>

Link Types

Various types of links you can create:

<!-- Absolute URLs -->
<a href="https://www.example.com">Website</a>

<!-- Relative URLs -->
<a href="about.html">About Page</a>

<!-- Email Links -->
<a href="mailto:info@example.com">Send Email</a>

<!-- Phone Links -->
<a href="tel:+1234567890">Call Us</a>

Link States and Styling

Links can have different states, which can be styled using CSS:

/* Unvisited link */
a:link {
  color: blue;
}

/* Visited link */
a:visited {
  color: purple;
}

/* Mouse over link */
a:hover {
  color: red;
}

/* Selected link */
a:active {
  color: green;
}

Page Bookmarks (Fragment Identifiers)

You can create links to specific sections within the same page using ID attributes:

<!-- Creating a bookmark -->
<h2 id="section1">Section 1</h2>

<!-- Link to the bookmark -->
<a href="#section1">Go to Section 1</a>

<!-- Link to a bookmark on another page -->
<a href="otherpage.html#section2">Go to Section 2 on another page</a>

HTML Link Attributes

Attribute Description Example
href Specifies the URL of the page the link goes to href="https://www.example.com"
target Specifies where to open the linked document target="_blank"
rel Specifies the relationship between current and linked document rel="nofollow"
title Provides extra information about the link title="Visit our homepage"
download Specifies that the target will be downloaded download="filename.pdf"

Link Examples

Link Type HTML Code Description
External Link <a href="https://www.example.com">Example Website</a> Links to an external website
Internal Link <a href="contact.html">Contact Us</a> Links to another page on the same website
Email Link <a href="mailto:info@example.com">Send Email</a> Opens the user's email client
Phone Link <a href="tel:+1234567890">Call Us</a> Initiates a phone call on mobile devices

Best Practices for Using Links

  • Use descriptive link text that clearly indicates where the link leads
  • Avoid using "click here" or "read more" as link text
  • Use the title attribute to provide additional information
  • Consider accessibility when creating links
  • Use target="_blank" for external links, but include rel="noopener noreferrer" for security
  • Keep link text concise but descriptive
  • Ensure links are visually distinguishable from regular text
