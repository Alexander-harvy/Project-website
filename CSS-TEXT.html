CSS Text Properties

CSS provides numerous properties to control the appearance and formatting of text on a webpage. These properties allow you to customize text color, alignment, decoration, spacing, and more.

Property Description
color Sets the color of text
text-align Specifies the horizontal alignment of text
text-decoration Adds decoration to text (underline, overline, line-through)
text-transform Controls text capitalization
text-indent Specifies indentation of the first line of text
line-height Sets the height of a line of text
letter-spacing Controls the spacing between characters
word-spacing Controls the spacing between words

Text Color

The color property sets the color of text:

h1 {
    color: blue;
}

p {
    color: #333333; /* Using hex code */
}

span {
    color: rgb(255, 0, 0); /* Using RGB */
}

Text Alignment

The text-align property specifies the horizontal alignment of text:

h1 {
    text-align: center;
}

p.left {
    text-align: left; /* Default value */
}

p.right {
    text-align: right;
}

p.justify {
    text-align: justify; /* Text is stretched to fit the width */
}

Text Decoration

The text-decoration property adds decoration to text:

h1 {
    text-decoration: underline;
}

p {
    text-decoration: none; /* No decoration */
}

a {
    text-decoration: none; /* Removes the default underline from links */
}

span {
    text-decoration: line-through;
}

div {
    text-decoration: overline;
}

Text Transformation

The text-transform property controls text capitalization:

h1 {
    text-transform: uppercase; /* All characters in uppercase */
}

p.lower {
    text-transform: lowercase; /* All characters in lowercase */
}

p.capital {
    text-transform: capitalize; /* First character of each word capitalized */
}

Text Spacing

CSS offers several properties to control text spacing:

p {
    line-height: 1.6; /* Controls the height of each line */
    letter-spacing: 2px; /* Spacing between characters */
    word-spacing: 5px; /* Spacing between words */
    text-indent: 30px; /* Indentation of the first line */
}

Text Shadow

The text-shadow property adds shadow to text:

h1 {
    text-shadow: 2px 2px 5px #888888;
    /* horizontal offset, vertical offset, blur radius, color */
}
