CSS Color Properties

Colors play a vital role in web design. CSS provides various ways to specify colors. Here are the most common color properties:

Property Description
color Sets the color of text
background-color Sets the background color of an element
border-color Sets the color of an element's border

Color Values

In CSS, colors can be specified in different ways:

1. Predefined Color Names

CSS has a list of predefined color names that can be used:

h1 {
    color: red;
}

p {
    color: blue;
}

div {
    background-color: green;
}

2. RGB Values

RGB stands for Red, Green, and Blue. Each parameter defines the intensity of the color with a value between 0 and 255:

h1 {
    color: rgb(255, 0, 0); /* Red */
}

p {
    color: rgb(0, 0, 255); /* Blue */
}

div {
    background-color: rgb(0, 128, 0); /* Green */
}

3. HEX Values

Hexadecimal color values are specified with: #RRGGBB, where RR (red), GG (green) and BB (blue) are hexadecimal values between 00 and FF:

h1 {
    color: #FF0000; /* Red */
}

p {
    color: #0000FF; /* Blue */
}

div {
    background-color: #008000; /* Green */
}

4. HSL Values

HSL stands for Hue, Saturation, and Lightness:

h1 {
    color: hsl(0, 100%, 50%); /* Red */
}

p {
    color: hsl(240, 100%, 50%); /* Blue */
}

div {
    background-color: hsl(120, 100%, 25%); /* Green */
}
