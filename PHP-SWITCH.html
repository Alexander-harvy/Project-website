PHP Switch Statement

The switch statement is used to perform different actions based on different conditions. It is similar to a series of if statements on the same expression, but it can be more efficient and readable, especially when you need to test an expression against many different values.

Basic Syntax


<?php
switch (expression) {
    case value1:
        // code to be executed if expression = value1
        break;
    case value2:
        // code to be executed if expression = value2
        break;
    ...
    default:
        // code to be executed if expression is different from all values
}
?>

How it works:

  • The switch expression is evaluated once
  • The value of the expression is compared with the values of each case
  • If there is a match, the associated code block is executed
  • The break statement prevents the code from running into the next case automatically
  • The default statement is used when no match is found

Simple Switch Example


<?php
$favColor = "blue";

switch ($favColor) {
    case "red":
        echo "Your favorite color is red!";
        break;
    case "blue":
        echo "Your favorite color is blue!";
        break;
    case "green":
        echo "Your favorite color is green!";
        break;
    default:
        echo "Your favorite color is neither red, blue, nor green!";
}
// Outputs: "Your favorite color is blue!"
?>

Switch Without Break Statement

If you omit the break statement, the code will continue to execute even if the expression matches a case. This is sometimes useful for specific logic where you want to run the same code for multiple cases.


<?php
$favColor = "blue";

switch ($favColor) {
    case "red":
        echo "Your favorite color is red! ";
    case "blue":
        echo "Your favorite color is blue! ";
    case "green":
        echo "Your favorite color is green! ";
    default:
        echo "You have a favorite color!";
}
// Outputs: "Your favorite color is blue! Your favorite color is green! You have a favorite color!"
?>

Multiple Cases - Same Action

You can have multiple case values for the same code block.


<?php
$day = 4;

switch ($day) {
    case 1:
    case 2:
    case 3:
    case 4:
    case 5:
        echo "It's a weekday";
        break;
    case 6:
    case 7:
        echo "It's the weekend";
        break;
    default:
        echo "Invalid day number";
}
// Outputs: "It's a weekday"
?>

Nested Switch Statements

You can nest switch statements inside other switch statements.


<?php
$color = "blue";
$shade = "dark";

switch ($color) {
    case "red":
        echo "Color is red. ";
        break;
    case "blue":
        echo "Color is blue. ";
        
        switch ($shade) {
            case "light":
                echo "Shade is light blue.";
                break;
            case "dark":
                echo "Shade is dark blue.";
                break;
            default:
                echo "Shade not specified.";
                break;
        }
        break;
    default:
        echo "Color not recognized.";
        break;
}
// Outputs: "Color is blue. Shade is dark blue."
?>

Switch vs. If-Else

Both switch and if-else statements can be used to make decisions in your code. Here's a comparison of the same logic implemented with both approaches:

Using If-Else:


<?php
$grade = "B";

if ($grade == "A") {
    echo "Excellent!";
} elseif ($grade == "B") {
    echo "Good job!";
} elseif ($grade == "C") {
    echo "Satisfactory";
} elseif ($grade == "D") {
    echo "You passed";
} elseif ($grade == "F") {
    echo "You failed";
} else {
    echo "Invalid grade";
}
// Outputs: "Good job!"
?>

Using Switch:


<?php
$grade = "B";

switch ($grade) {
    case "A":
        echo "Excellent!";
        break;
    case "B":
        echo "Good job!";
        break;
    case "C":
        echo "Satisfactory";
        break;
    case "D":
        echo "You passed";
        break;
    case "F":
        echo "You failed";
        break;
    default:
        echo "Invalid grade";
}
// Outputs: "Good job!"
?>

When to Use Switch Instead of If-Else

  • When you are comparing a single variable against many different values
  • When you have many conditions to check against a single variable
  • For better readability when you have 4+ conditions on the same variable
  • When performance matters (switch can be faster in some programming languages)

Limitations of Switch

  • Switch only works with simple equality comparisons (==, not >, <, <=, >=, etc.)
  • Each case value must be a simple type (integer, float, string) or a constant expression
  • You cannot use complex conditions or multiple variables in the case statements

Real-World Example: Day of the Week


<?php
// Get the current day of the week (1 = Monday, 7 = Sunday)
$dayOfWeek = date("N");

switch ($dayOfWeek) {
    case 1:
        echo "Today is Monday. Start of the work week!";
        break;
    case 2:
        echo "Today is Tuesday. Keep going!";
        break;
    case 3:
        echo "Today is Wednesday. Halfway there!";
        break;
    case 4:
        echo "Today is Thursday. Almost there!";
        break;
    case 5:
        echo "Today is Friday. TGIF!";
        break;
    case 6:
        echo "Today is Saturday. Enjoy your weekend!";
        break;
    case 7:
        echo "Today is Sunday. Relax before the new week!";
        break;
    default:
        echo "Invalid day!";
        break;
}
?>

Best Practices

  • Always include the break statement unless you specifically want to fall through to the next case
  • Always include a default case for handling unexpected values
  • Keep the switch statement readable by using proper indentation
  • Order your cases in a logical sequence (e.g., alphabetical, numerical, or most common cases first)
  • Consider using if-else statements instead when comparing different variables or using complex conditions
