CSS Display Property

The CSS display property specifies how an element is displayed on a webpage. It is one of the most important properties for controlling layout and determining how elements interact with each other.

Value Description
block Displays an element as a block element (starts on a new line and takes up the full width)
inline Displays an element as an inline element (on the same line, taking only necessary width)
inline-block Displays an element as an inline-level block container
flex Displays an element as a block-level flex container
grid Displays an element as a block-level grid container
none The element is completely removed from the page

Display: Block

Block-level elements start on a new line and take up the full width available:

div {
    display: block; /* Default for div elements */
    width: 300px; /* Will respect width property */
    height: 100px; /* Will respect height property */
    margin: auto; /* Can be centered using margin: auto */
    background-color: lightblue;
}

span {
    display: block; /* Changes inline span to block */
    width: 50%; /* Width can be set */
}

Examples of elements that are block-level by default: div, h1-h6, p, form, header, footer, section.

Display: Inline

Inline elements do not start on a new line and only take up as much width as necessary:

span {
    display: inline; /* Default for span elements */
    width: 300px; /* Width property has no effect */
    height: 100px; /* Height property has no effect */
    margin-top: 50px; /* Top and bottom margins have no effect */
    background-color: lightgreen;
}

div {
    display: inline; /* Changes block div to inline */
}

Examples of elements that are inline by default: span, a, img, em, strong, i, small, code.

Display: Inline-Block

Inline-block elements are placed inline (on the same line) but behave as block elements:

span {
    display: inline-block;
    width: 100px; /* Width is respected */
    height: 100px; /* Height is respected */
    margin: 10px; /* All margins are respected */
    background-color: lightcoral;
}

.nav-item {
    display: inline-block;
    padding: 10px 20px;
    margin-right: 5px;
    background-color: #f1f1f1;
}

Display: None

Elements with display: none are completely removed from the page layout:

.hidden {
    display: none; /* Element will not be displayed and does not take up space */
}

/* Different from visibility: hidden, which hides the element but still takes up space */
.invisible {
    visibility: hidden;
}

Display: Flex

Flex containers enable a flexible layout without using float or positioning:

.container {
    display: flex;
    justify-content: space-between; /* Horizontal alignment */
    align-items: center; /* Vertical alignment */
}

See the Flexbox page for more details on flex containers and flex items.

Display: Grid

Grid containers enable grid-based layouts:

.container {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr; /* Three equal columns */
    grid-gap: 20px; /* Gap between grid items */
}

See the Grid page for more details on grid containers and grid items.
