PHP Operators

Operators are used to perform operations on variables and values. PHP divides the operators into the following groups:

  • Arithmetic operators
  • Assignment operators
  • Comparison operators
  • Increment/Decrement operators
  • Logical operators
  • String operators
  • Array operators
  • Conditional assignment operators

Arithmetic Operators

The PHP arithmetic operators are used with numeric values to perform common arithmetical operations.

Operator Name Example Result
+ Addition $x + $y Sum of $x and $y
- Subtraction $x - $y Difference of $x and $y
* Multiplication $x * $y Product of $x and $y
/ Division $x / $y Quotient of $x and $y
% Modulus $x % $y Remainder of $x divided by $y
** Exponentiation $x ** $y Result of raising $x to the $y'th power 

<?php
$x = 10;
$y = 3;

echo $x + $y;  // Outputs: 13
echo $x - $y;  // Outputs: 7
echo $x * $y;  // Outputs: 30
echo $x / $y;  // Outputs: 3.333...
echo $x % $y;  // Outputs: 1
echo $x ** $y; // Outputs: 1000
?>

Assignment Operators

The PHP assignment operators are used to write values to variables.

Operator Example Same As
= $x = $y $x = $y
+= $x += $y $x = $x + $y
-= $x -= $y $x = $x - $y
*= $x *= $y $x = $x * $y
/= $x /= $y $x = $x / $y
%= $x %= $y $x = $x % $y 

<?php
$x = 10;
echo $x; // Outputs: 10

$x += 5;
echo $x; // Outputs: 15

$x -= 3;
echo $x; // Outputs: 12

$x *= 2;
echo $x; // Outputs: 24

$x /= 4;
echo $x; // Outputs: 6
?>

Comparison Operators

The PHP comparison operators are used to compare two values (number or string).

Operator Name Example Result
== Equal $x == $y True if $x is equal to $y
=== Identical $x === $y True if $x is equal to $y, and they are of the same type
!= Not equal $x != $y True if $x is not equal to $y
<> Not equal $x <> $y True if $x is not equal to $y
!== Not identical $x !== $y True if $x is not equal to $y, or they are not of the same type
> Greater than $x > $y True if $x is greater than $y
< Less than $x < $y True if $x is less than $y
>= Greater than or equal to $x >= $y True if $x is greater than or equal to $y
<= Less than or equal to $x <= $y True if $x is less than or equal to $y
<=> Spaceship $x <=> $y Returns -1, 0, or 1 depending on if $x is less than, equal to, or greater than $y 

<?php
$x = 5;
$y = "5";

var_dump($x == $y);   // bool(true)
var_dump($x === $y);  // bool(false)
var_dump($x != $y);   // bool(false)
var_dump($x !== $y);  // bool(true)
var_dump($x < $y);    // bool(false)
var_dump($x > $y);    // bool(false)
var_dump($x <=> $y); // int(0)

$y = 10;
var_dump($x <=> $y); // int(-1)
var_dump($y <=> $x); // int(1)
?>

Increment/Decrement Operators

The PHP increment/decrement operators are used to increment/decrement a variable's value.

Operator Name Description
++$x Pre-increment Increments $x by one, then returns $x
$x++ Post-increment Returns $x, then increments $x by one
--$x Pre-decrement Decrements $x by one, then returns $x
$x-- Post-decrement Returns $x, then decrements $x by one 

<?php
$x = 5;
echo ++$x; // Outputs: 6
echo $x;   // Outputs: 6

$x = 5;
echo $x++; // Outputs: 5
echo $x;   // Outputs: 6

$x = 5;
echo --$x; // Outputs: 4
echo $x;   // Outputs: 4

$x = 5;
echo $x--; // Outputs: 5
echo $x;   // Outputs: 4
?>

Logical Operators

The PHP logical operators are used to combine conditional statements.

Operator Name Example Result
and And $x and $y True if both $x and $y are true
or Or $x or $y True if either $x or $y is true
xor Xor $x xor $y True if either $x or $y is true, but not both
&& And $x && $y True if both $x and $y are true
|| Or $x || $y True if either $x or $y is true
! Not !$x True if $x is not true 

<?php
$x = true;
$y = false;

var_dump($x and $y);  // bool(false)
var_dump($x or $y);   // bool(true)
var_dump($x xor $y);  // bool(true)
var_dump($x && $y);   // bool(false)
var_dump($x || $y);   // bool(true)
var_dump(!$x);        // bool(false)
?>

String Operators

PHP has two operators that are specially designed for strings.

Operator Name Example Result
. Concatenation $txt1 . $txt2 Concatenation of $txt1 and $txt2
.= Concatenation assignment $txt1 .= $txt2 Appends $txt2 to $txt1 

<?php
$txt1 = "Hello";
$txt2 = " World!";

echo $txt1 . $txt2; // Outputs: Hello World!

$txt1 .= $txt2;
echo $txt1; // Outputs: Hello World!
?>

Array Operators

The PHP array operators are used to compare arrays.

Operator Name Example Result
+ Union $x + $y Union of $x and $y
== Equality $x == $y True if $x and $y have the same key/value pairs
=== Identity $x === $y True if $x and $y have the same key/value pairs in the same order and of the same types
!= Inequality $x != $y True if $x is not equal to $y
<> Inequality $x <> $y True if $x is not equal to $y
!== Non-identity $x !== $y True if $x is not identical to $y 

<?php
$x = array("a" => "red", "b" => "green");
$y = array("c" => "blue", "d" => "yellow");

$z = $x + $y; // Union of $x and $y
print_r($z);

$x = array("a" => "red", "b" => "green");
$y = array("b" => "green", "a" => "red");

var_dump($x == $y);   // bool(true)
var_dump($x === $y);  // bool(false)
var_dump($x != $y);   // bool(false)
var_dump($x !== $y);  // bool(true)
?>

Conditional Assignment Operators

The PHP conditional assignment operators are used to set a value depending on conditions.

Operator Name Example Result
?: Ternary $x = expr1 ? expr2 : expr3 Returns the value of $x. The value of $x is expr2 if expr1 = TRUE. The value of $x is expr3 if expr1 = FALSE
?? Null coalescing $x = expr1 ?? expr2 Returns the value of $x. The value of $x is expr1 if expr1 exists, and is not NULL. If expr1 does not exist, or is NULL, the value of $x is expr2. 

<?php
// Ternary operator
$age = 20;
$status = ($age >= 18) ? "Adult" : "Minor";
echo $status; // Outputs: Adult

// Null coalescing operator
$name = $_GET['name'] ?? 'Guest';
echo $name; // Outputs: Guest if $_GET['name'] is not set
?>

Operator Precedence

Operator precedence determines the grouping of terms in an expression. This affects how an expression is evaluated.

Precedence Operator Description
1 clone new clone and new
2 ** Exponentiation
3 ++ -- ~ (int) (float) (string) (array) (object) (bool) @ Increment/decrement and type casting
4 ! Logical NOT
5 * / % Multiplication, division, modulus
6 + - . Addition, subtraction, concatenation
7 << >> Bitwise shift
8 < <= > >= Comparison
9 == != === !== <> <=> Comparison
10 & Bitwise AND
11 ^ Bitwise XOR
12 | Bitwise OR
13 && Logical AND
14 || Logical OR
15 ?? Null coalescing
16 ?: Ternary
17 = += -= *= **= /= .= %= &= |= ^= <<= >>= Assignment
18 and Logical AND
19 xor Logical XOR
20 or Logical OR 

<?php
// Example showing operator precedence
$result = 5 + 3 * 2; // Multiplication has higher precedence than addition
echo $result; // Outputs: 11

// Use parentheses to force precedence
$result = (5 + 3) * 2;
echo $result; // Outputs: 16
?>
