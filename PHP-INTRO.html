Introduction to PHP

What is PHP?

PHP (PHP: Hypertext Preprocessor) is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML.

Unlike client-side languages like JavaScript that run in the browser, PHP code is executed on the server, generating HTML which is then sent to the client. The client receives the results of running that script, but has no knowledge of the underlying code.

What Can PHP Do?

  • Generate dynamic page content
  • Create, open, read, write, delete, and close files on the server
  • Collect form data
  • Send and receive cookies
  • Add, delete, modify data in your database
  • Control user-access
  • Encrypt data

Why PHP?

  • PHP runs on various platforms (Windows, Linux, Unix, Mac OS X, etc.)
  • PHP is compatible with almost all servers used today (Apache, IIS, etc.)
  • PHP supports a wide range of databases
  • PHP is free to download and use
  • PHP is easy to learn and runs efficiently on the server side

PHP File Extension

PHP files have the extension ".php"

A PHP file normally contains HTML tags, and some PHP scripting code. Below is an example of a simple PHP file, with a PHP script that outputs "Hello World!":


<!DOCTYPE html>
<html>
<body>

<h1>My First PHP Page</h1>

<?php
echo "Hello World!";
?>

</body>
</html>

How PHP Works

When a visitor opens a page containing PHP code, the server processes the PHP commands and sends the resulting HTML to the browser. This processing flow is what makes PHP a server-side language.

PHP Development Environment

To develop with PHP, you'll need:

  1. A web server (like Apache)
  2. PHP installed on your server
  3. A database system (like MySQL) if you plan to work with databases
  4. A text editor (like VSCode, Sublime Text, etc.) for writing code

You can install these individually or use an all-in-one package like:

  • XAMPP (for Windows, Mac, Linux)
  • WAMP (for Windows)
  • MAMP (for Mac)
  • LAMP (for Linux)

Your First PHP Script

Let's create a basic "Hello World" script in PHP:


<?php
    // This is a PHP comment
    echo "Hello, World!"; // This prints text to the screen
?>

Save this code in a file named "hello.php" and place it in your web server's document root directory. Then access it via your browser by visiting "http://localhost/hello.php".
