PHP Syntax

Basic PHP Syntax

A PHP script can be placed anywhere in the document. A PHP script starts with <?php and ends with ?>:


<?php
// Your PHP code goes here
?>

The default file extension for PHP files is ".php".

A PHP file normally contains HTML tags and some PHP scripting code.

PHP Case Sensitivity

In PHP, keywords (like if, else, while, echo, etc.), classes, functions, and user-defined functions are not case-sensitive.

However, all variable names are case-sensitive!


<?php
// These are all equivalent
ECHO "Hello World!<br>";
echo "Hello World!<br>";
EcHo "Hello World!<br>";

// But these are different variables
$color = "red";
echo "My car is " . $color . "<br>";
echo "My house is " . $COLOR . "<br>"; // Undefined variable $COLOR
echo "My boat is " . $coLOR . "<br>"; // Undefined variable $coLOR
?>

PHP Statements

In PHP, each code statement must end with a semicolon (;). The semicolon is a statement terminator.


<?php
echo "Hello World!";
echo "This is a PHP tutorial";
?>

PHP Comments

Comments in PHP are lines that are not executed as part of the program. Their purpose is to make the code more readable and to leave notes for yourself or other developers.

PHP supports several ways to comment:


<?php
// This is a single-line comment

# This is also a single-line comment

/*
This is a multiple-lines comment block
that spans over multiple
lines
*/

// You can also use comments to leave out parts of code
// echo "This line will not be executed";
?>

PHP in HTML

PHP can be embedded directly within HTML, which is what makes it so powerful for web development. You can switch between HTML and PHP modes:


<!DOCTYPE html>
<html>
<body>

<h1>My First PHP Page</h1>

<?php
echo "Hello World!";
?>

<p>This is HTML content after the PHP section.</p>

<?php
echo "Another PHP section!";
?>

</body>
</html>

PHP Output Functions

PHP provides several ways to output data to the browser:

  • echo - Can output one or more strings
  • print - Can output one string
  • print_r() - Used to print human-readable information about a variable
  • var_dump() - Outputs more detailed information including data type and length

<?php
// Using echo
echo "Hello World!";
echo "Multiple", " strings", " can", " be", " outputted";

// Using print
print "Hello World!";

// Using print_r() for arrays
$colors = array("red", "green", "blue");
print_r($colors);

// Using var_dump()
$x = 5;
var_dump($x);
?>

PHP Short Tags

Short opening tags are an alternative syntax for PHP's standard opening tags. Instead of <?php, you can use <?. However, this is not recommended for compatibility reasons.


<? 
echo "This is a short tag (not recommended)"; 
?>

<?php
echo "This is the standard tag (recommended)";
?>

Note: The short tag format may be disabled in some server configurations. For maximum compatibility, always use the standard <?php opening tag.

Practical Example: A Simple PHP Script

Let's create a simple PHP script that displays the current date and time:


<!DOCTYPE html>
<html>
<head>
    <title>Date and Time Example</title>
</head>
<body>
    <h1>The current date and time is:</h1>
    
    <?php
    // Get and display the current date/time
    echo date("Y-m-d H:i:s");
    ?>
    
    <p>This time is generated by PHP's date() function.</p>
</body>
</html>

Save this code in a file named "datetime.php" and place it in your web server's document root. When accessed via a browser, it will display the current server date and time.
