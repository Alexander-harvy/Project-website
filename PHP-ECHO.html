PHP Output - Echo and Print

PHP Output

PHP has several functions for outputting data to the browser. The two most common methods are echo and print.

The Echo Statement

The echo statement can be used with or without parentheses: echo or echo().


<?php
echo "<h2>PHP is Fun!</h2>";
echo "Hello world!<br>";
echo "I'm about to learn PHP!<br>";
echo "This ", "string ", "was ", "made ", "with multiple parameters.";
?>

Displaying Variables with Echo

The following example shows how to output text and variables with the echo statement:


<?php
$txt1 = "Learn PHP";
$txt2 = "W3Schools.com";
$x = 5;
$y = 4;

echo "<h2>" . $txt1 . "</h2>";
echo "Study PHP at " . $txt2 . "<br>";
echo $x + $y;
?>

The Print Statement

The print statement can be used with or without parentheses: print or print().


<?php
print "<h2>PHP is Fun!</h2>";
print "Hello world!<br>";
print "I'm about to learn PHP!";
?>

Displaying Variables with Print

The following example shows how to output text and variables with the print statement:


<?php
$txt1 = "Learn PHP";
$txt2 = "W3Schools.com";
$x = 5;
$y = 4;

print "<h2>" . $txt1 . "</h2>";
print "Study PHP at " . $txt2 . "<br>";
print $x + $y;
?>

Echo vs Print

The differences between echo and print are small:

  • echo has no return value while print has a return value of 1 so it can be used in expressions
  • echo can take multiple parameters (although such usage is rare) while print can take one argument
  • echo is marginally faster than print
Feature echo print
Return value None 1
Parameters Multiple Single
Speed Faster Slower
Usage Statement Function-like

Other Output Functions

PHP provides other output functions that are useful for debugging and displaying information:

print_r()

The print_r() function is used to display structured information about variables in a more human-readable form:


<?php
$fruits = array("apple", "banana", "orange");
print_r($fruits);
?>

Output:


Array
(
    [0] => apple
    [1] => banana
    [2] => orange
)

var_dump()

The var_dump() function displays structured information about variables that includes their type and value:


<?php
$x = 5;
$y = "Hello";
$z = true;
$a = array("red", "green", "blue");

var_dump($x);
var_dump($y);
var_dump($z);
var_dump($a);
?>

Output:


int(5)
string(5) "Hello"
bool(true)
array(3) {
  [0]=>
  string(3) "red"
  [1]=>
  string(5) "green"
  [2]=>
  string(4) "blue"
}

Practical Example: Creating a Simple HTML Template

In this example, we'll use echo statements to generate an HTML template with dynamic content:


<!DOCTYPE html>
<html>
<head>
    <title>My PHP Website</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 20px;
        }
        .header {
            background-color: #4CAF50;
            color: white;
            padding: 20px;
            text-align: center;
        }
        .content {
            padding: 20px;
        }
    </style>
</head>
<body>
    <?php
    // Define variables
    $siteName = "My Dynamic PHP Site";
    $userName = "John Doe";
    $currentDate = date("F j, Y");
    
    // Output the header section
    echo "<div class='header'>";
    echo "<h1>$siteName</h1>";
    echo "<p>Welcome, $userName! Today is $currentDate</p>";
    echo "</div>";
    
    // Output the content section
    echo "<div class='content'>";
    echo "<h2>About This Page</h2>";
    echo "<p>This is a simple example showing how to use PHP's echo statement to generate dynamic HTML content.</p>";
    
    // Generate a list using a loop
    echo "<h3>My Favorite Programming Languages:</h3>";
    echo "<ul>";
    $languages = array("PHP", "JavaScript", "Python", "Java", "C++");
    foreach ($languages as $language) {
        echo "<li>$language</li>";
    }
    echo "</ul>";
    
    echo "</div>";
    ?>
</body>
</html>

Save this code in a file named "template.php" and place it in your web server's document root. When accessed via a browser, it will display a complete HTML page with dynamic content generated by PHP.
