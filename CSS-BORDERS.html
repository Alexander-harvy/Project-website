CSS Border Properties

CSS borders allow you to define the style, width, and color of an element's border. You can apply borders to any HTML element.

Property Description
border-width Sets the width of the border
border-style Sets the style of the border
border-color Sets the color of the border
border Shorthand property for setting width, style, and color in one declaration
border-radius Adds rounded corners to elements

Border Width

The border-width property specifies the width of the border. You can use px, pt, cm, em, or keywords (thin, medium, thick):

p {
    border-style: solid;
    border-width: 5px;
}

h1 {
    border-style: solid;
    border-width: thin; /* thin, medium, or thick */
}

Border Style

The border-style property specifies the type of border to display:

/* Available styles */
p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}
p.double {border-style: double;}
p.groove {border-style: groove;}
p.ridge {border-style: ridge;}
p.inset {border-style: inset;}
p.outset {border-style: outset;}
p.none {border-style: none;}
p.hidden {border-style: hidden;}

Border Color

The border-color property is used to set the color of the border:

p {
    border-style: solid;
    border-color: red;
}

h1 {
    border-style: solid;
    border-color: #FF0000; /* HEX value */
}

div {
    border-style: solid;
    border-color: rgb(255, 0, 0); /* RGB value */
}

Border Shorthand

The border property is a shorthand property for setting all border properties at once:

p {
    border: 5px solid red;
    /* Order: width style color */
}

Individual Sides

You can also specify borders for individual sides:

p {
    border-top: 5px solid red;
    border-right: 3px dotted blue;
    border-bottom: 2px dashed green;
    border-left: 1px solid black;
}

Border Radius

The border-radius property adds rounded corners to elements:

div {
    border: 2px solid red;
    border-radius: 10px; /* All corners equally rounded */
}

.custom-rounded {
    border: 2px solid blue;
    border-radius: 10px 20px 30px 40px; /* top-left, top-right, bottom-right, bottom-left */
}
