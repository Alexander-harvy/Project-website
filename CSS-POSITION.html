CSS Position Property

The position property specifies the type of positioning method used for an element. It allows you to precisely place elements on a webpage and control how they interact with other elements.

Value Description
static Default value. Elements are positioned according to the normal flow of the document
relative Element is positioned relative to its normal position
absolute Element is positioned relative to its nearest positioned ancestor
fixed Element is positioned relative to the browser window
sticky Element is positioned based on the user's scroll position

Position: Static

Static positioned elements are not affected by the top, bottom, left, and right properties. This is the default position value:

div {
    position: static;
    /* top, right, bottom, left properties have no effect */
}

Position: Relative

Relative positioned elements are offset relative to their normal position:

.box {
    position: relative;
    top: 20px; /* Move down 20px from normal position */
    left: 30px; /* Move right 30px from normal position */
    background-color: lightblue;
}

The element still occupies the original space in the document flow, but appears offset from that position.

Position: Absolute

Absolute positioned elements are removed from the normal document flow and positioned relative to the nearest positioned ancestor (not static):

.container {
    position: relative; /* Creates a positioning context for absolute children */
    width: 500px;
    height: 300px;
    border: 2px solid black;
}

.box {
    position: absolute;
    top: 50px; /* 50px from the top of the positioned parent */
    right: 30px; /* 30px from the right of the positioned parent */
    width: 100px;
    height: 100px;
    background-color: coral;
}

If no positioned ancestor exists, it positions relative to the document body.

Position: Fixed

Fixed positioned elements are removed from the normal document flow and positioned relative to the viewport (browser window):

.header {
    position: fixed;
    top: 0; /* Fixed at the top of the viewport */
    left: 0;
    width: 100%;
    background-color: #333;
    color: white;
    padding: 10px;
    z-index: 100; /* Controls stacking order */
}

Fixed elements remain in the same position even when the page is scrolled.

Position: Sticky

Sticky positioned elements are positioned based on the user's scroll position:

.nav {
    position: sticky;
    top: 0; /* Sticks to the top when scrolled to this position */
    background-color: #f1f1f1;
    padding: 10px;
    z-index: 50;
}

A sticky element toggles between relative and fixed, depending on the scroll position. It's positioned relative until a given offset position is met in the viewport, then it "sticks" in place (like position: fixed).

Z-Index

The z-index property specifies the stack order of positioned elements (which element should be placed in front of, or behind, the others):

.box1 {
    position: absolute;
    z-index: 2; /* Higher value appears on top */
    /* Other properties */
}

.box2 {
    position: absolute;
    z-index: 1; /* Lower value appears behind */
    /* Other properties */
}

Z-index only works on positioned elements (position: relative, absolute, fixed, or sticky).
