HTML Basics - Getting Started with HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. Understanding the basics is the foundation for web development. Let's explore the fundamental concepts of HTML.

Document Structure

Every HTML document has a basic structure:

<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>
  Content goes here...
</body>
</html>

This structure includes the document type declaration, html, head, and body elements that form the foundation of every HTML page.

HTML Syntax

HTML uses tags to define elements:

  • Tags are enclosed in angle brackets: <tagname>
  • Most tags come in pairs: <tagname>content</tagname>
  • The closing tag has a forward slash: </tagname>
  • Some tags are self-closing: <tagname />

Understanding this syntax is essential for writing valid HTML code.

HTML Elements

Basic elements you'll use frequently:

  • <h1> to <h6> - headings
  • <p> - paragraphs
  • <a> - links
  • <img> - images
  • <ul>, <ol>, <li> - lists
  • <div> - container

These elements form the building blocks of your web pages.

HTML Elements Explained

Element Purpose
<!DOCTYPE html> Defines document type
<html> Root element of page
<head> Contains meta-information
<title> Sets page title in browser tab
<body> Contains visible page content

Your First HTML Page

  • Create a new file in your text editor
  • Copy the basic HTML structure shown above
  • Add some content inside the body tags: 
    <body>
  <h1>My First Heading</h1>
  <p>My first paragraph.</p>
</body>
  • Save the file with a .html extension (e.g., "index.html")
  • Open the file in a web browser to see your first webpage

Tips for HTML Beginners

  • Always use proper indentation to make your code readable
  • Make sure all your tags are properly closed
  • Use lowercase for tag names for better compatibility
  • Validate your HTML using online validators
  • Start simple and gradually add more complex elements
