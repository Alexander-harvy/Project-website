PHP Loops

Loops are used to execute a block of code repeatedly. In PHP, there are four types of loops:

  • while - loops through a block of code as long as the specified condition is true
  • do...while - loops through a block of code once, and then repeats the loop as long as the specified condition is true
  • for - loops through a block of code a specified number of times
  • foreach - loops through a block of code for each element in an array or object

The while Loop

The while loop executes a block of code as long as the specified condition is true.


<?php
// Syntax
while (condition) {
    // code to be executed
}

// Example
$x = 1;

while ($x <= 5) {
    echo "The number is: $x 
";
    $x++;
}

/* Output:
The number is: 1
The number is: 2
The number is: 3
The number is: 4
The number is: 5
*/
?>

The do...while Loop

The do...while loop will always execute the block of code once, then check the condition, and repeat the loop as long as the condition is true.


<?php
// Syntax
do {
    // code to be executed
} while (condition);

// Example
$x = 1;

do {
    echo "The number is: $x 
";
    $x++;
} while ($x <= 5);

/* Output:
The number is: 1
The number is: 2
The number is: 3
The number is: 4
The number is: 5
*/
?>

The key difference between while and do...while is that do...while will always execute the code block at least once, even if the condition is false from the start.


<?php
// Example showing the difference
$x = 6;

// while loop - condition is false from the start, so nothing executes
while ($x <= 5) {
    echo "This will not be printed (while)";
    $x++;
}

// do...while loop - executes once even though condition is false
do {
    echo "This will be printed once (do...while)";
    $x++;
} while ($x <= 5);
?>

The for Loop

The for loop is used when you know in advance how many times you want to execute a block of code.


<?php
// Syntax
for (initialization; condition; increment) {
    // code to be executed
}

// Example
for ($x = 0; $x <= 10; $x += 2) {
    echo "The number is: $x 
";
}

/* Output:
The number is: 0
The number is: 2
The number is: 4
The number is: 6
The number is: 8
The number is: 10
*/
?>

The for loop parameters:

  • initialization - initializes a counter variable (executed once at the beginning)
  • condition - evaluated for each loop iteration, if true, the loop continues; if false, the loop ends
  • increment - increases (or decreases) the counter value after each iteration

The foreach Loop

The foreach loop is designed to work specifically with arrays and objects. It loops through each element in an array or each property in an object.


<?php
// Syntax for arrays
foreach ($array as $value) {
    // code to be executed
}

// Syntax for associative arrays
foreach ($array as $key => $value) {
    // code to be executed
}

// Example with indexed array
$colors = array("red", "green", "blue", "yellow");

foreach ($colors as $value) {
    echo "$value 
";
}

/* Output:
red
green
blue
yellow
*/

// Example with associative array
$age = array("Peter"=>"35", "Ben"=>"37", "Joe"=>"43");

foreach ($age as $key => $value) {
    echo "$key is $value years old 
";
}

/* Output:
Peter is 35 years old
Ben is 37 years old
Joe is 43 years old
*/
?>

Break and Continue

The break and continue statements allow you to control the flow of loops.

The break Statement

The break statement is used to "jump out" of a loop, stopping it from executing any further.


<?php
// Example of break
for ($x = 0; $x < 10; $x++) {
    if ($x == 4) {
        break;
    }
    echo "The number is: $x 
";
}

/* Output:
The number is: 0
The number is: 1
The number is: 2
The number is: 3
*/
?>

The continue Statement

The continue statement is used to "skip" the current iteration of a loop, and continue with the next iteration.


<?php
// Example of continue
for ($x = 0; $x < 10; $x++) {
    if ($x == 4) {
        continue;
    }
    echo "The number is: $x 
";
}

/* Output:
The number is: 0
The number is: 1
The number is: 2
The number is: 3
The number is: 5
The number is: 6
The number is: 7
The number is: 8
The number is: 9
*/
?>

Nested Loops

Loops can be nested within each other. A common use case is to iterate through multi-dimensional arrays.


<?php
// Nested for loop example - creating a multiplication table
echo "";
for ($i = 1; $i <= 5; $i++) {
    echo "";
    for ($j = 1; $j <= 5; $j++) {
        echo "";
    }
    echo "";
}
echo "
" . ($i * $j) . "
";

/* Output:
A 5x5 multiplication table showing products from 1x1 to 5x5
*/
?>

Best Practices for Loops

  • Choose the right loop for the job: use for when you know the number of iterations, foreach for arrays, while for condition-based iterations.
  • Avoid infinite loops by ensuring that your loop condition will eventually be false.
  • Keep loop bodies as simple as possible; consider extracting complex logic into functions.
  • For large datasets, be cautious about memory usage, especially when dealing with large arrays in foreach loops.
  • Use the break statement to exit a loop early when a specific condition is met.
  • Use the continue statement to skip iterations that don't need processing.
  • When processing HTML output in loops, consider using output buffering to improve performance.
