PHP Constants

Constants are like variables except that once they are defined they cannot be changed or undefined. A constant is an identifier (name) for a simple value.

Create a PHP Constant

To create a constant, use the define() function:

Syntax: define(name, value, case-insensitive)


<?php
define("GREETING", "Welcome to W3Schools.com!");
echo GREETING;
?>

Case-Insensitive Constants

By default, constants are case-sensitive. To make them case-insensitive, set the third parameter to true:


<?php
define("GREETING", "Welcome to W3Schools.com!", true);
echo greeting; // outputs "Welcome to W3Schools.com!"
?>

Constants are Global

Constants are automatically global and can be used across the entire script.


<?php
define("GREETING", "Welcome to W3Schools.com!");

function myTest() {
    echo GREETING;
}
 
myTest();
?>

PHP const Keyword

You can also create constants using the const keyword (since PHP 5.3):


<?php
const MYCAR = "Volvo";
echo MYCAR;
?>

const vs define()

Feature const define()
Case sensitivity Always case-sensitive Can be case-insensitive
Global scope Yes Yes
Can be used in class Yes No
Value type Scalar (int, float, string, bool) Any valid PHP expression
Namespace support Yes Yes (PHP 5.3+)

Magic Constants

PHP has nine predefined constants that change depending on where they are used:


<?php
echo __LINE__;      // Current line number of the file
echo __FILE__;      // Full path and filename of the file
echo __DIR__;       // Directory of the file
echo __FUNCTION__;  // Function name
echo __CLASS__;     // Class name
echo __TRAIT__;     // Trait name
echo __METHOD__;    // Class method name
echo __NAMESPACE__; // Namespace name
?>

Practical Example: Configuration File

Here's an example of using constants for application configuration:


<?php
// config.php - Configuration file with constants
define("DB_HOST", "localhost");
define("DB_USER", "root");
define("DB_PASS", "password");
define("DB_NAME", "my_database");
define("SITE_TITLE", "My PHP Application");
define("MAX_LOGIN_ATTEMPTS", 3);
define("DEBUG_MODE", true);

// main.php - Application file
require_once 'config.php';

function connectToDatabase() {
    $conn = new mysqli(DB_HOST, DB_USER, DB_PASS, DB_NAME);
    if ($conn->connect_error) {
        die("Connection failed: " . $conn->connect_error);
    }
    return $conn;
}

echo "<h1>" . SITE_TITLE . "</h1>";
echo "<p>Connecting to database: " . DB_NAME . "</p>";

if (DEBUG_MODE) {
    echo "<div style='background:#eee;padding:10px;'>";
    echo "<strong>Debug Info:</strong><br>";
    echo "File: " . __FILE__ . "<br>";
    echo "Line: " . __LINE__ . "<br>";
    echo "</div>";
}
?>
